# User states


Copy page


Copy page


| User state | Description |
|----|----|
| `ACTIVE` | The user is active. |
| `RECOVERY_STARTED` | <a href="/cli/recover-users" class="link">Account recovery</a> has been started for the user. |
| `RECOVERY_ACCEPTED` | The user has created their new account password and is waiting to be <a href="https://support.1password.com/recovery/#complete-recovery" class="link" target="_blank" rel="noreferrer">confirmed again by an administrator</a>. |
| `SUSPENDED` | The user is suspended. |
| `TRANSFER_STARTED` | The user has been provisioned, but hasn’t set up their account. |
| `TRANSFER_SUSPENDED` | The user was provisioned and didn’t set up their account before they were deprovisioned. |
