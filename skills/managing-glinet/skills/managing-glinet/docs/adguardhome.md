# AdGuard Home (Built-in)

Network-wide ad and tracker blocking. Once enabled, covers all devices connected to the router.

## Supported Models
GL-E5800, GL-MT5000, GL-MT3600BE, GL-BE6500, GL-BE9300, GL-BE3600, GL-X2000, GL-B3000, GL-MT6000, GL-AX1800, GL-X3000, GL-XE3000, GL-MT3000, GL-AXT1800, GL-A1300, GL-MT2500/A, GL-AP1300, GL-S1300.

**Not supported:** GL-SFT1200 (Opal), GL-MT1300 (Beryl), GL-E750 (Mudi), GL-AR750S (Slate), GL-XE300 (Puli), GL-X750 (Spitz), GL-MT300N-V2 (Mango), GL-AR300M (Shadow), GL-B1300, GL-X300B.

## Setup

1. Web Admin Panel → **APPLICATIONS → AdGuard Home**
2. Enable toggle → Click **Apply**
3. Click **Settings Page** to access AdGuard Home dashboard for advanced configuration

## Configuration

### Handle Client Requests
When enabled, processes DNS queries directly through AdGuard Home — shows per-client DNS activity. **Note:** May prevent domain-based VPN policies from working.

### Advanced Settings (via AdGuard dashboard)
- Filter rule management
- Custom blocklist URLs
- DNS configuration
- Query log and statistics

## Dashboard
Real-time monitoring: DNS query stats, blocked domains, query sources.
