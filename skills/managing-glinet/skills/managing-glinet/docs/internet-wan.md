# Internet & WAN Setup

Access via **INTERNET** in the web admin panel.

## Status Diagram
- Solid green line = active connectivity
- Dashed gray = disconnected
- White line + yellow exclamation = connected but no internet
- Shows status of: AdGuard Home, IPv6, VPN, Tor, WiFi (2.4G/5G), connected device counts

## Connection Types

### Ethernet
Wired connection via network cable to WAN port.

### Repeater
Extend an existing WiFi network. Router connects wirelessly to upstream AP and rebroadcasts.

### Tethering
Share mobile device internet via USB or Bluetooth connection.

### Cellular
Built-in LTE/5G modem (model dependent — Spitz, Puli, Mudi series).

## Setup Wizard
Available since firmware v4.7 via the book icon in top-right corner. Guides through initial configuration for all connection types. Also available in the GL.iNet mobile app.

## WiFi QR Codes
Hover over enabled WiFi icons to show QR codes for quick mobile connection.
