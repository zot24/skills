# Guides Overview

This page serves as a navigation hub for integrating Honcho memory capabilities across various platforms and frameworks.

## AI Assistants

- **Claude Code**: Long-term memory that survives context wipes, session restarts, and project switches
- **MCP Server**: Adds memory to Claude Desktop, Cursor, Windsurf, Cline, and other MCP clients
- **Hermes Agent**: Cross-session memory for Nous Research's framework
- **OpenClaw**: Memory across WhatsApp, Telegram, Discord, Slack, and additional channels
- **Agent Zero**: Persistent memory as a plugin for the Agent Zero framework

## Platform Connectors

- **Discord Bot**: Bots that retain user context across conversations
- **Telegram Bot**: Bots with persistent user understanding
- **Gmail**: Import email threads with participant and conversation data
- **Granola**: Ingest meeting transcripts including speaker turns
- **Paperclip**: Integrate memory into companies, agents, issues, and documents
- **Reachy Mini**: Embodied voice robots with long-term memory

## Agent Frameworks

- **LangGraph**: Persistent memory for LangGraph workflows
- **CrewAI**: Memory integration for CrewAI agent orchestration
- **Zo Computer**: Memory for Zo Computer agents
- **n8n**: Memory integration for n8n automation workflows
