# The Honcho Dashboard

## Overview

Honcho is a platform designed to help developers build stateful agents without worrying about infrastructure. The service provides a cloud-based dashboard for managing agent instances with comprehensive features for data organization, API management, and monitoring.

## Getting Started

### Step 1: Account Creation
Users begin by visiting app.honcho.dev to create an account. Those with existing team members using Honcho can request an invitation to their organization, while new users receive a prompt to establish a fresh organization.

### Step 2: Instance Activation
After organization setup, users navigate to the Billing section to add a payment method. The system then provisions the Honcho instance automatically, with deployment progress visible on the Instance Status page.

### Step 3: API Key Management
The API Keys section permits creation of authentication tokens with varying permission levels -- from full instance access to scope-limited keys restricted to specific Workspaces, Peers, or Sessions.

### Step 4: API Testing
The API Playground offers a Postman-like interface for testing endpoints, validating integrations, and examining real-time responses with comprehensive request/response logging.

## Core Features

**Workspaces**: Containers for organizing Honcho data where users create workspaces and manage entities.

**Peers & Sessions**: Within workspaces, users manage Peers (entities) and Sessions (interactions), with capabilities to add multiple Peers to Sessions and edit metadata.

**Monitoring**: The Performance page displays usage metrics, health analytics, API response times, and endpoint statistics.

**Webhooks**: Integration capability enabling webhook creation and management for event-driven workflows.

**Team Management**: The Members page allows organization administrators to control team access and assign admin permissions.

## Dashboard Sections

- Performance analytics and health monitoring
- Peer utilities including message search and chat functionality
- Session utilities for message management and context generation
- Workspace management with comprehensive data tables
- Webhook configuration and monitoring
