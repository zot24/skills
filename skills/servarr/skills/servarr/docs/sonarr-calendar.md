# <a href="#calendar" class="toc-anchor">¶</a> Calendar

The calendar shows you recently aired episodes, as well as upcoming episodes.

# <a href="#ical-feed" class="toc-anchor">¶</a> iCal Feed

- Sonarr can provide your calendar as an iCal feed. This will contain the previous <a href="https://github.com/Sonarr/Sonarr/blob/main/src/Sonarr.Api.V3/Calendar/CalendarFeedController.cs" class="is-external-link">7 days</a> and the next <a href="https://github.com/Sonarr/Sonarr/blob/main/src/Sonarr.Api.V3/Calendar/CalendarFeedController.cs" class="is-external-link">28 days</a>

> Google Calendar has internal issues that results in it no longer updating. This is a Google issue and not a Sonarr issue. It can often be resolved by removing and re-adding the calendar.
