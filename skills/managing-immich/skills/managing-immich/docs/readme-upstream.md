# Immich

High performance self-hosted photo and video management solution.

- **Website**: https://immich.app
- **Documentation**: https://docs.immich.app
- **Demo**: https://demo.immich.app (demo@immich.app / demo)
- **License**: AGPLv3

## Features

- Photo/video upload and backup with duplication prevention
- Multi-user support with album sharing
- Advanced search (metadata, objects, faces, CLIP)
- Raw format support and EXIF metadata viewing
- Facial recognition and clustering
- Global map functionality
- Partner sharing and public sharing options
- Offline support (mobile)
- Mobile apps for iOS and Android

## Important Notice

Maintain a 3-2-1 backup plan for media preservation. The database contains only metadata — media files must be backed up independently.
